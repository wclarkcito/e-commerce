# e-commerce

## this is a command line application that manages the back end database. Through the application you will be able to View, Create, Update, and Delete Products, Tags and Categories using Insomnia.

# Instructions

## will need to open up the command line in server.js, and run npm run start to connect the server. Once that has been completed all previously specified actions will be performed inside of insomnia. Please refer to the included instructional video.
